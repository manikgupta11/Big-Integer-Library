# Implementation of Java Big Integer Library in C++:

**Functions it can perform:**
- Add 2 BigIntegers
- Multiply 2 BigIntegers
- Fast Exponentiation of 2 BigIntegers
- GCD of 2 BigIntegers
- Factorial of a BigInteger
